# RAG Knowledge Assistant

An open-source Retrieval-Augmented Generation (RAG) application for semantic document search and question answering.

---

## Vision

The goal of this project is to build a modern RAG application from scratch while understanding every component of the pipeline.

Instead of relying on large frameworks, we want to implement the core ideas ourselves to better understand how Retrieval-Augmented Generation works.

This project is intended as

- a learning project
- a portfolio project for AI Engineering
- an open-source application

---

## Planned Features

- PDF upload
- Intelligent document chunking
- Semantic embeddings
- Vector database
- Semantic search
- Chat with documents
- Source citations
- Multi-document support
- Hybrid retrieval
- Optional local LLM support
